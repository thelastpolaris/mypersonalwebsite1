# Портфолио Артема Федоскина

Это одностраничный статичный сайт-портфолио для Артема Федоскина, Product Manager из Астаны. Проект создан с использованием HTML и стилизован с помощью Tailwind CSS.

## Стек технологий

-   **HTML5**
-   **Tailwind CSS** - Утилитарный CSS-фреймворк, подключенный через CDN.
-   **JavaScript (Vanilla)** - Для добавления небольшой интерактивности.

## Запуск

Поскольку это полностью статичный сайт, для его просмотра не требуется сборка или установка зависимостей.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd <название папки>
    ```

2.  **Откройте файл в браузере:**
    Просто откройте файл `index.html` в вашем любимом веб-браузере.

## Развертывание

Вы можете развернуть этот сайт на любом статическом хостинге (например, GitHub Pages, Netlify, Vercel), просто загрузив содержимое этого репозитория. Для GitHub Pages достаточно включить его в настройках репозитория для `main` ветки.
