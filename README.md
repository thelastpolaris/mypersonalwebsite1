# Портфолио Артема Федоскина

Это одностраничный сайт-портфолио для Артема Федоскина, Product Manager из Астаны. Проект создан с использованием React, TypeScript и стилизован с помощью Tailwind CSS.

## Стек технологий

-   **React** - Библиотека для создания пользовательских интерфейсов.
-   **TypeScript** - Типизированный JavaScript для повышения надежности кода.
-   **Tailwind CSS** - Утилитарный CSS-фреймворк для быстрой стилизации.
-   **Vite** - Современный и быстрый сборщик для веб-приложений.

## Локальная разработка

Для запуска проекта на вашем компьютере выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd <название папки>
    ```

2.  **Установите зависимости:**
    Убедитесь, что у вас установлен [Node.js](https://nodejs.org/). Затем выполните команду:
    ```bash
    npm install
    ```

3.  **Запустите сервер для разработки:**
    ```bash
    npm run dev
    ```
    Эта команда запустит локальный сервер. Откройте браузер и перейдите по адресу, указанному в терминале (обычно `http://localhost:5173`).

## Сборка для продакшена

Чтобы создать оптимизированную версию приложения для развертывания, выполните команду:
```bash
npm run build
```
Это создаст папку `dist` в корне проекта, содержащую статические файлы вашего приложения, готовые к развертыванию на любом хостинге.

## Развертывание на GitHub Pages

Этот проект настроен для автоматического развертывания на GitHub Pages при каждом пуше в ветку `main`.

### Конфигурация

1.  **Название репозитория**: В файле `vite.config.ts` необходимо указать правильное название вашего репозитория в свойстве `base`. Например, если ваш репозиторий находится по адресу `https://github.com/user/my-repo`, то `base` должен быть `'/my-repo/'`.

    ```typescript
    // vite.config.ts
    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'

    export default defineConfig({
      // ЗАМЕНИТЕ 'artem-fedoskin-portfolio' НА НАЗВАНИЕ ВАШЕГО РЕПОЗИТОРИЯ
      base: '/artem-fedoskin-portfolio/', 
      plugins: [react()],
    })
    ```

2.  **Настройки GitHub Pages**: В настройках вашего репозитория на GitHub, перейдите в раздел "Pages". В качестве источника (`Source`) выберите ветку `gh-pages` и папку `/(root)`. Ветка `gh-pages` будет создана и обновлена автоматически при первом успешном выполнении workflow.
